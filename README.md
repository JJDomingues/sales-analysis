# Sales Analysis

SQL + Python data analysis project using SQLite.

## Features
- Database created with SQLite
- Queries with SELECT, WHERE, ORDER BY, GROUP BY
- Python connected to SQLite via sqlite3

## Tech Stack
- Python 3.x
- SQLite3
- DB Browser for SQLite

## How to Run
```bash
python3 sales_analysis.py
```
